## Project

Knob — a native macOS menu bar app for offline push-to-talk dictation using whisper.cpp. Swift + SwiftUI, targeting macOS 14+ on Apple Silicon. Single-user tool (no accounts, no cloud, no Electron).

## Architecture

The app has five components wired in a linear pipeline:

1. **Global Hotkey** — `CGEvent` tap captures Right Option key-down/key-up globally (requires Accessibility permission)
2. **Audio Capture** — `AVAudioEngine` input tap records 16kHz mono Float32 PCM into an in-memory `[Float]` buffer (requires Microphone permission)
3. **whisper.cpp Inference** — C library linked via bridging header, called through `whisper_init_from_file` / `whisper_full` / `whisper_full_get_segment_text`. Model: `ggml-small.en.bin` stored in `~/Library/Application Support/Knob/models/`
4. **Text Output** — save clipboard → copy transcription → synthetic Cmd+V via `CGEvent` → restore clipboard after ~200ms
5. **Menu Bar UI** — SwiftUI `MenuBarExtra` with `.menu` style showing status, model name, and quit

## Build Setup

- Xcode project generated by **XcodeGen** from `project.yml` (source of truth). The `.xcodeproj` is gitignored.
- whisper.cpp included as a **git submodule** at `vendor/whisper.cpp`, built as a static library via CMake with `GGML_METAL=ON`, `GGML_METAL_EMBED_LIBRARY=ON`, `WHISPER_COREML=OFF`
- CMake build runs as an Xcode pre-build script phase (`Knob/Scripts/build-whisper.sh`)
- Code-signed with personal development certificate (no App Store)
- Bundle ID: `com.mrkcnd.knob`

### Build Commands

```
xcodegen generate
bash Knob/Scripts/build-whisper.sh
xcodebuild -project Knob.xcodeproj -scheme Knob -configuration Debug -arch arm64 build
```

## Key whisper.cpp Config

- Sampling strategy: `WHISPER_SAMPLING_GREEDY`
- `n_threads`: 4, `no_context`: true, `single_segment`: true
- `suppress_blank`: true, `suppress_non_speech_tokens`: true
- Metal GPU acceleration enabled for matrix multiplications

## Constraints

- Push-to-talk only (no toggle mode)
- Recording: min 0.5s (discard shorter), max 30s (auto-stop)
- Post-processing: trim whitespace and leading space only — no punctuation/capitalization correction
- Target: <2s inference for 10s audio, <3s end-to-end for 5s speech
- Resource budget: <200MB RAM idle, <1GB during inference

## Implementation Phases

1. **CLI proof of concept** — build whisper.cpp, Swift CLI that records + transcribes
2. **Menu bar shell** — minimal SwiftUI menu bar app with Accessibility check and CGEvent tap
3. **Integration** — wire audio capture, inference, and clipboard paste together
4. **Polish** — duration guards, first-launch model check, error handling
